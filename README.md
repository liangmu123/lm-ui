# @class_liangmu/lm-ui
- 基于element-ui 封装的业务组件库
- 支持按需加载

## Install
```shell
npm install @class_liangmu/lm-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import LMUI from '@class_liangmu/lm-ui'

Vue.use(LMUI)

// or
import {
  Button
  // ...
} from '@class_liangmu/lm-ui'

Vue.use(Button)
```
